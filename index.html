Muamua

มั่วมั่ว แอพพลิเคชั่น social media ใหม่ ของเด็กมหาลัย

What is Muamua?

-- ฟังก์ชั่น แอพมั่วมั่ว --

- โพสต์ข้อความโดยไม่ต้อง login
- ดูข้อความที่น่าสนใจของคนอื่นในรัศมี 6 ไมล์
- ข้อความที่มี vote เท่ากับ -6 จะโดนลบโดยอัตโนมัติ
- แอบดูข้อความในรั้วมหาลัยชื่อดังต่างๆ
- upvote / downvote ข้อความ
- เเจ้งข้อความที่ไม่เหมาะสม

Rules / Regulations

-- กฎหมายของ มั่วมั่ว --

- ห้ามโพสต์ ข้อความที่ เกี่ยวกับยาเสพติด ล่วงละเมิดผู้อื่น ดูหมิ่น สถาบันชาติ ศาสนา พระมหากษัตริย์
- Admin สามารถลบข้อความของผู้ใช้ได้โดยไม่ต้องขออนุญาต์
- Admin สามารถ ban ผู้ใช้ได้โดยไม่ต้องขออนุญาต
- มั่วมั่วไม่มีส่วนเกี่ยวข้องกับข้อความไดไดทั้งสิ้น
